# Gently Used Lada ($LADA)

A simple, single-page website for the Solana meme coin **Gently Used Lada ($LADA)**.

The project raises funds (via creator fees) to support the purchase of armored vehicles and logistical aid in Ukraine through @peterktodd.

## Live Demo

Coming soon via GitHub Pages.

## Features

- Fully responsive single-page design
- Copyable contract address
- Charity Tracker section with animated counters
- Clean Soviet / Lada meme aesthetic
- Built with Tailwind CSS (via CDN)
- No build step required

## Getting Started

### View locally

Just open `index.html` in your browser.

### Deploy on GitHub Pages (Recommended)

1. Create a new repository on GitHub (e.g. `lada-site`)
2. Upload the `index.html` file to the root of the repo
3. Go to **Settings → Pages**
4. Under "Build and deployment", set:
   - Source: **Deploy from a branch**
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`
5. Click **Save**
6. Your site will be live at: `https://yourusername.github.io/lada-site`
